To‑Do List

Click a task to mark it as done

Overview

A lightweight task manager demonstrating list creation, completion toggles, and simple persistence patterns suitable for small projects.

Stack

  • HTML, CSS, JavaScript

Features

  • Add and remove tasks
  • Click a task to mark as done
  • Clean, responsive UI

How it works

  1. Enter a task and submit to add it to the list.
  2. Click on a task item to toggle its completed state.
  3. Use the remove action to delete items you no longer need.
